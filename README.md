# Cookis-Garden-Helper-Reloaded-Reloaded
Further development of the popular Cookie Clicker mod Cookis Garden Helper Reloaded (steam version)

Added features:
1. Option to require all plants to be mature for auto harvest base on cps
2. Option to require all intended slots to be empty for auto plant
3. Option to automatically freeze garden when all cps-based plants are mature
4. Changed ratio of growing vs mature plants that trigger auto switch soil from 1:1 to 2:1
